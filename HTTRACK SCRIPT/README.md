# Edison Lite drupaldemomigrations

This site is based on Vue JavaScript Framework.

Read the [Edison Lite documentation](https://docs.platforms.pfizer/edison/docs).

## Project setup guide
```
npm install
```

### Compiles and watches for development
```
npm run dev
```

### Serve with hot reload at localhost:8080
```
npm run hot
```

### Compiles for production
```
npm run prod
```

### Lints and fixes files
```
npm run lint
```
